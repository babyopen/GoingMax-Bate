## 三、API 详细说明

### 3.1 HTML 转义（防 XSS）

```js
CommonString.escapeHtml('<script>alert("xss")</script>');
// &lt;script&gt;alert(&quot;xss&quot;)&lt;/script&gt;

CommonString.escapeAttr('a"b\'c');  // a&quot;b&#39;c

CommonString.unescapeHtml('&lt;b&gt;hi&lt;/b&gt;');  // <b>hi</b>
```

### 3.2 数字格式化

```js
// 号码格式化（参数化）
CommonString.formatNum(5);                          // '05'（默认 1-49）
CommonString.formatNum(5, { min: 1, max: 100 });    // '05'
CommonString.formatNum(0, { min: 1, max: 49 });     // '00'（无效值）
CommonString.formatNum(123, { width: 5 });          // '00123'

// 千分位
CommonString.thousands(1234567);  // '1,234,567'

// 中文大写金额
CommonString.toChineseMoney(1234.56);  // '壹仟贰佰叁拾肆元伍角陆分'

// 补零
CommonString.padZero(5);       // '05'
CommonString.padZero(5, 4);    // '0005'
```

### 3.3 名称生成

```js
// 唯一名称
CommonString.ensureUniqueName('方案', [{name:'方案'}, {name:'方案 (2)'}]);
// '方案 (3)'

// 重命名时排除自身
CommonString.ensureUniqueName('方案', list, 0);

// 默认名称
CommonString.nextDefaultName([{name:'方案1'}, {name:'方案2'}], '方案');
// '方案3'

CommonString.nextDefaultName([{name:'订单1'}, {name:'订单3'}], '订单');
// '订单4'（跳过已删除的 2）
```

### 3.4 字符串操作

```js
CommonString.truncate('这是一段很长的文本', 5);          // '这是一段很...'
CommonString.truncate('这是一段很长的文本', 5, '…');    // '这是一段很…'

CommonString.capitalize('hello');   // 'Hello'

CommonString.camelToKebab('myClassName');   // 'my-class-name'
CommonString.kebabToCamel('my-class-name'); // 'myClassName'
```

## 五、跨项目复用

`common-string.js` 是**最易跨项目复用**的文件之一：
- ✅ 零项目特定依赖
- ✅ 零 DOM / 零状态 / 零存储
- ✅ 纯函数工具集
- ✅ 跨项目 1:1 复制即可

**常见跨项目场景**：
- 电商网站：toChineseMoney（金额大写）/ truncate（商品描述）/ padZero（订单号）
- 内容平台：escapeHtml（评论 XSS 防护）/ truncate（摘要）/ kebabToCamel（路由转 state 名）
- 后台系统：ensureUniqueName（重名策略）/ nextDefaultName（默认用户名）
- 任何项目：escapeHtml（XSS 必防）

## 六、安全建议

⚠️ **HTML 转义是 XSS 防护的必备步骤**，所有 `innerHTML` 插入前必须经过 `escapeHtml`：

```js
// ❌ 危险
el.innerHTML = userInput;

// ✅ 安全
el.innerHTML = CommonString.escapeHtml(userInput);
```
