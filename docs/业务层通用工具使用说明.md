# 业务层通用工具使用说明（2026-06-20 新增）

## 工具列表

| 文件 | 函数 | 用途 |
|---|---|---|
| [business-common-sort.js](../business/business-common-sort.js) | `sortByNumberDesc` | 按数字字段降序 |
| | `sortByNumberAsc` | 按数字字段升序 |
| | `sortByCountDesc` | 按 count 字段降序 |
| | `sortEntriesByValueDesc` | Object.entries 后按 value 降序 |
| | `sortByZodiacName` | 生肖名本地化排序 |
| [business-common-time.js](../business/business-common-time.js) | `formatRelative` | "X 分钟/小时/天前" 相对时间 |
| | `formatAbsolute` | "YYYY-MM-DD HH:mm" 绝对时间 |
| | `_formatAbsolute` | 内部辅助 |

## 使用方法

### 挂载（一次性）

由于 `index.html` 是只读文件，需手动在 `index.html` 末尾追加：

```html
<!-- 在 business/business-main.js 之前 -->
<script src="business/business-common-sort.js"></script>
<script src="business/business-common-time.js"></script>
```

### 在视图层使用

**示例 1**：将 [views/view-analysis-full.js](../views/analysis/view-analysis-full.js) L121 的排序下沉

```js
// ❌ 原代码（视图层业务计算）
var entries = Object.entries(dataObj).sort(function(a, b) { return b[1] - a[1]; });

// ✅ 优化后（调用业务层）
var entries = BusinessCommonSort.sortEntriesByValueDesc(Object.entries(dataObj));
```

**示例 2**：将 [views/view-zodiac-giong.js](../views/zodiac/view-zodiac-giong.js) L400 的排序下沉

```js
// ❌ 原代码
statZones.sort(function(a, b) { return b.count - a.count; });

// ✅ 优化后
var sorted = BusinessCommonSort.sortByCountDesc(statZones);
```

**示例 3**：将 [views/view-zodiac-main.js](../views/zodiac/view-zodiac-main.js) L51-62 的时间格式化下沉

```js
// ❌ 原代码
var updateTime = new Date(timestamp);
if (...) {
  ageText = Math.floor(ageHours) + '小时前';
}

// ✅ 优化后
var ageText = BusinessCommonTime.formatRelative(timestamp);
```

## 不影响现有代码

按"只能新增"原则，本工具**不替换**任何现有代码。
视图层可以选择性迁移，或保持原状。
