## 三、统一返回格式

所有 `Validator.*` 方法返回：
```js
{ valid: boolean, error: string|null, value: any }
```

调用方统一处理：
```js
var r = CommonValidator.Validator.validateNumber(5, { min: 1, max: 49 });
if (!r.valid) {
  Toast.show(r.error);
  return;
}
// r.value 即为合法值
```

## 四、API 详细说明

### 4.1 `validateHistoryData(data, itemValidator?)`

**跨项目亮点**：可自定义每条记录的验证函数。

```js
// 默认（验证彩票格式：expect + 7 位 openCode）
var r1 = CommonValidator.Validator.validateHistoryData(historyData);

// 自定义（验证电商订单）
var r2 = CommonValidator.Validator.validateHistoryData(orders, function(item) {
  return item && item.orderId && item.amount > 0;
});
```

### 4.2 `validateNumber(num, opts?)`

参数化范围，跨项目可用：

```js
// 默认 1-49（彩票）
CommonValidator.Validator.validateNumber(5);

// 1-100（游戏）
CommonValidator.Validator.validateNumber(50, { min: 1, max: 100 });

// 0-9999（订单号）
CommonValidator.Validator.validateNumber('1234', { min: 0, max: 9999 });
```

### 4.3 `validateEnum(value, validList, fieldName?)`

通用化原 `validateZodiac`：

```js
// 生肖
CommonValidator.Validator.validateEnum(zodiac, CONFIG.ANALYSIS.ZODIAC_ALL, '生肖');

// 订单状态
CommonValidator.Validator.validateEnum(status, ['待付款', '已发货', '已完成'], '订单状态');

// 用户角色
CommonValidator.Validator.validateEnum(role, ['admin', 'user', 'guest'], '角色');
```

### 4.4 `validatePeriod(period, min?, max?)`

```js
CommonValidator.Validator.validatePeriod(period);         // 默认 1-500
CommonValidator.Validator.validatePeriod(period, 1, 100); // 自定义 1-100
```

### 4.5 `safeExecute(fn, args, context?)`

```js
var r = CommonValidator.Validator.safeExecute(
  function(input) { return heavyCompute(input); },
  historyData,
  '计算特码'
);
if (!r.success) {
  console.error(r.error);
  return r.error;
}
return r.result;
```

### 4.6 🆕 `validateRequired(str, fieldName?)`

```js
CommonValidator.Validator.validateRequired(username, '用户名');
// { valid: false, error: '用户名不能为空', value: '' }
```

### 4.7 🆕 `validatePositive(value)`

```js
CommonValidator.Validator.validatePositive(amount);
// { valid: true, error: null, value: 100 }
// { valid: false, error: '必须为正数', value: 0 }
```

### 4.8 🆕 `validateArray(arr, fieldName?)`

```js
CommonValidator.Validator.validateArray(items, '选中的生肖');
```

## 六、跨项目复用

`common-validator.js` 是**最易跨项目复用**的文件之一：
- ✅ 零项目特定依赖
- ✅ 零 DOM / 零状态 / 零存储
- ✅ 纯函数 + 工具集
- ✅ 跨项目 1:1 复制即可

**跨项目使用步骤**：
1. 复制 `core/common-validator.js` 到目标项目
2. 在 index.html 加载
3. 直接调用 `CommonValidator.*` 即可
