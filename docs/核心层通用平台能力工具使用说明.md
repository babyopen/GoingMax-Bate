## 三、API 详细说明

### 3.1 设备 / 平台判断

```js
CommonPlatform.isMobile();        // true / false
CommonPlatform.isIOS();           // true / false
CommonPlatform.isAndroid();       // true / false
CommonPlatform.isWechat();        // true / false（微信内置浏览器）
CommonPlatform.isMiniProgram();   // true / false（小程序 web-view）

// 使用示例
if (CommonPlatform.isIOS()) {
  // iOS 专属处理
}

if (CommonPlatform.isWechat() || CommonPlatform.isMiniProgram()) {
  // 微信 / 小程序环境处理
}
```

### 3.2 安全区 / 视口

```js
// 通用 CSS 变量
CommonPlatform.getCssVar('--primary-color');       // 数字
CommonPlatform.getCssVar('--top-bar-height', someEl);

// 顶部安全区（默认 --safe-top）
CommonPlatform.getSafeTop();                       // 默认
CommonPlatform.getSafeTop('--ios-safe-top');       // 自定义变量名

// 底部安全区
CommonPlatform.getSafeBottom();                    // 默认 --safe-bottom

// 视口尺寸
var size = CommonPlatform.getViewportSize();
// { width: 375, height: 812 }
```

### 3.3 复制到剪贴板（3 级降级）

```js
// 基础用法
await CommonPlatform.copyToClipboard('要复制的文本');

// 自定义提示
await CommonPlatform.copyToClipboard(text, {
  successMsg: '已复制到剪贴板',
  errorMsg: '复制失败，请手动复制'
});

// 提供 fallback
await CommonPlatform.copyToClipboard(text, {
  fallback: function(text) {
    // 打开 modal 让用户手动复制
    showCopyModal(text);
  }
});
```

**降级策略**：
1. `navigator.clipboard.writeText`（HTTPS / localhost）
2. `document.execCommand('copy')` + 隐藏 textarea
3. `opts.fallback` 回调 / Toast 错误提示

### 3.4 网络状态

```js
CommonPlatform.isOnline();   // true / false

// 监听变化（返回取消监听函数）
var unbind = CommonPlatform.watchNetwork(
  function() { Toast.show('网络已恢复'); },
  function() { Toast.show('网络已断开'); }
);

// 组件销毁时
unbind();
```

### 3.5 浏览器能力检测

```js
CommonPlatform.hasLocalStorage();    // false（隐私模式可能 false）
CommonPlatform.hasSessionStorage();
CommonPlatform.hasPromise();
CommonPlatform.hasAsyncAwait();

// 使用示例
if (!CommonPlatform.hasLocalStorage()) {
  // 降级到 cookie 或内存存储
}
```

## 五、跨项目复用

`common-platform.js` 是**跨项目通用工具**：
- ✅ 仅依赖浏览器原生 API（navigator / window / document / Toast）
- ✅ Toast 调用已做 `typeof Toast !== 'undefined'` 容错（Toast 不存在不报错）
- ✅ 所有环境判断都安全（navigator 不存在时返回 false / 默认值）
- ✅ 跨项目 1:1 复制即可

**常见跨项目场景**：
- 移动端 H5：isMobile / isIOS / isAndroid / getSafeTop / getSafeBottom
- 小程序 web-view：isMiniProgram / watchNetwork
- 微信生态：isWechat + 微信 JS-SDK 调用
- 任何项目：copyToClipboard（3 级降级方案）
- PWA：watchNetwork + hasLocalStorage（离线检测）

## 七、安全 / 容错说明

所有方法都做了容错：

| 方法 | 容错策略 |
|---|---|
| `isMobile / isIOS / ...` | navigator 不存在 → 返回 false |
| `getCssVar / getSafeTop` | document 不存在 → 返回 0 |
| `copyToClipboard` | Toast 不存在 → 不调用 Toast |
| `isOnline` | navigator 不存在 → 返回 true |
| `watchNetwork` | window 不存在 → 返回空 unbind 函数 |
| `hasLocalStorage` | try-catch 包裹 setItem |
