## 六、每期标准执行 SOP

### 步骤 1：确定起点

- 取最新一期开奖号码 **N**
- 若 **N** 为外围号 → 映射至挂靠主链节点 **M**，起点 = **M**
- 若 **N** 为主链号 → 起点 = **N**

### 步骤 2：生成双链顺位

- 从起点沿**新版主链**向后取 **前4顺位**（不含起点）
- 从起点沿**旧版主链**向后取 **前4顺位**（不含起点）

### 步骤 3：构建候选池

- 候选池 = 新版4顺位 ∪ 旧版4顺位 ∪ 变盘号 **06**

### 步骤 4：降权过滤

- 统计近12期和近11期各号码频次
- 剔除 12期≥3次 且 11期≠2次 的号码
- 恢复 12期≥3次 且 11期=2次 的号码资格

### 步骤 5：映射前导号码

- 将上期号码、两期前号码按挂靠体系映射为主链节点

### 步骤 6：评分排序

- 对候选池中每个号码，按5.1\~5.10逐项打分
- 检查多热压制是否触发
- 计算总分，降序排列

### 步骤 7：24期冷热替换

- 检查主推前4名中是否有24期冷号（≤3次）
- 若有，用备选中24期热度最高的链内号码替换
- 重新确认前4名

### 步骤 8：输出结果

- **主推4码**：评分前4（已替换冷号）
- **备选2码**：评分第5-6名
- **排除清单**：降权未解除号码

## 七、融合统一推荐公式

text

```
候选池 = 新版4顺位 ∪ 旧版4顺位 ∪ {06}

评分 = 链位置分 + 属性分 + 11期趋势分(24期修正) 
       + 连开分 + 破冰分 + 24期热度分 
       + 多热压制分（如触发）
       + 对子热度分(映射后) + 三步链匹配分(映射后) + 循环链匹配分(映射后)

主推4码 = 评分前4名（若含24期冷号则替换为备选热号）
备选2码 = 评分第5-6名
排除 = 降权未解除号
```

## 八、方案维护声明

1. 本方案为最终完善版，所有规则永久固定，包含挂靠映射、模块1融合、24期热度背景板、多热压制等全部优化。
2. 每期严格按 SOP 执行，自动输出主推4码+备选2码，无需人工干预。
3. 极端异常行情（连续5期以上偏离主链）仅调整冷热阈值，不改动核心规则。
4. 所有历史对子、三步链、循环链统计均以挂靠映射后的主链节点为准。
5. 所有推演仅供历史数据走势分析参考。
